== Description ==

Si vous avez beaucoup d'articles planifiés, en brouillon ou autre, ce plugin vous propose d'ajouter une couleur sur le fond de la ligne dans l'interface d'administration. 

Ce sera beaucoup plus simple pour gérer vos publications !

== Installation ==

1. Upload `StatusPostColor.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
